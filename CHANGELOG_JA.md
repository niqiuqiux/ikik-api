# 変更履歴

ikik-api の重要な変更はこのファイルに記録します。

このプロジェクトでは、release tag を公開済みバージョンの正本として扱います。このファイルには、ユーザーおよび運用者向けの変更点とアップグレードノートを記載します。

## 1.0.0

### 新機能

- プロジェクト名を ikik-api とし、セルフホスト AI ゲートウェイおよびサブスクリプション管理向けに整理しました。
- OpenAI 互換ゲートウェイを追加し、複数プロバイダーへのリクエスト転送とアカウント型スケジューリングに対応しました。
- アカウントプール管理、API Key グループ、クォータ制御、利用記録、課金メタデータを追加しました。
- ユーザーサブスクリプション、チャージ、引換コード、招待報酬、ショップ/カードキー機能を追加しました。
- ユーザー、アカウント、チャネル、決済、モデレーション、リスクイベント、データ管理、システム設定の管理機能を追加しました。
- tag ベースのリリースフローを追加し、Docker イメージ、アーカイブ、GitHub Releases に対応しました。

### 改善

- リバースプロキシ、PostgreSQL、Redis、リクエストボディ制限、CDN キャッシュ回避、バックアップに関する本番デプロイ説明を追加しました。
- シークレット、管理画面アクセス、決済認証情報、ストレージ認証情報、モデレーション認証情報、メール認証情報に関するセキュリティ説明を追加しました。
- 英語、中国語、日本語の README を追加しました。

### 修正

- ent のネストされたトランザクション、Redis テスト分離、usage log のウォレット明細クエリ順序に関するリポジトリ統合テスト互換性を修正しました。

### アップグレードノート

- 既存デプロイでマイグレーションまたはアップグレードを行う前に、PostgreSQL をバックアップしてください。
- `deploy/config.example.yaml` を確認し、本番認証情報を git にコミットしないでください。
- Nginx を使用し、クライアントがアンダースコアを含むヘッダーを送信する場合は、Nginx の `http` ブロックで `underscores_in_headers on;` を有効化してください。
