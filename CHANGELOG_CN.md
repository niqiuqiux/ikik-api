# 更新日志

ikik-api 的重要变更会记录在这里。

本项目以 release tag 作为已发布版本的事实来源。本文件只记录面向用户和运维人员的变更与升级说明。

## 1.0.0

### 新功能

- 项目更名为 ikik-api，面向自托管 AI 网关与订阅管理场景。
- 添加 OpenAI 兼容网关能力，支持多供应商请求转发和账号型调度。
- 添加账号池管理、API Key 分组、额度控制、使用记录和计费元数据。
- 添加用户订阅、充值、兑换码、邀请奖励和商城/卡密流程。
- 添加用户、账号、渠道、支付、风控、风险事件、数据管理和系统设置等后台运营能力。
- 添加基于 tag 的发布流程，支持 Docker 镜像、归档包和 GitHub Releases。

### 优化

- 补充生产部署说明，包括反向代理、PostgreSQL、Redis、请求体限制、CDN 缓存绕过和备份。
- 补充安全说明，覆盖密钥、后台访问、支付凭据、存储凭据、风控凭据和邮件凭据。
- 添加英文、中文、日文三份 README 文档。

### 修复

- 修复仓储层集成测试兼容问题，包括 ent 嵌套事务、Redis 测试隔离和 usage log 钱包明细查询顺序。

### 升级说明

- 对已有部署执行迁移或升级前，请先备份 PostgreSQL。
- 检查 `deploy/config.example.yaml`，不要把生产凭据提交到 git。
- 如果使用 Nginx 且客户端会发送带下划线的请求头，请在 Nginx 的 `http` 块中启用 `underscores_in_headers on;`。
