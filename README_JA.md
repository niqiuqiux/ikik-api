# ikik-api

ikik-api is a modified AI API gateway and subscription management platform based on Sub2API.

For setup and development notes, see:

- [English README](README.md)
- [Chinese README](README_CN.md)

This repository is intended for self-hosting and further customization. Do not publish production secrets, private server configuration, API keys, payment credentials, or database backups.

ikik-api keeps attribution to the upstream Sub2API project. Preserve upstream license requirements when redistributing this repository.
